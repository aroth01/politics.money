About {{ site_title }}

Making Utah campaign finance data accessible and transparent

Our Mission

This site aims to provide better data analytics about campaign finance in Utah. The current site at disclosures.utah.gov is a bit dated. And it just shows PDF report pages and doesn't really provide anything else than raw data. To help show better insights on money, hopefully we can show some of the motiviations in Utah politics. There are api endpoints for some of this data, but all API access is rate limited and may be refused at any time for any reason. If you do choose to use the API, please have some method of contact in the User-Agent header so that we can reach out if there are any issues.

What We Do

Data Collection

We automatically collect and parse campaign finance disclosure reports from disclosures.utah.gov, the official Utah Lieutenant Governor's Office disclosure portal.

Easy Navigation

Browse reports by PACs, contributors, and expenditures. Use our search feature to find specific organizations, people, or transactions.

Visualizations

Interactive charts and graphs help you understand contribution patterns, spending trends, and financial flows at a glance.

Out-of-State Analysis

See where out-of-state contributions are coming from with our interactive map and analysis tools.

Data Source

All data on this site comes from official campaign finance disclosure reports filed with the Utah Lieutenant Governor's Office and published at disclosures.utah.gov.

We update our database regularly to ensure you have access to the latest filings. The last update time is shown in the footer of every page.

Disclaimer

This is an independent project and is not affiliated with the Utah Lieutenant Governor's Office or any government agency. While we strive for accuracy, this site is provided for informational purposes only. For official records, please refer to disclosures.utah.gov.
The data contained on this site is not subject to UCPA. We do not accept request to remove records. Even if we did, your info is still publically stored online by the State of Utah. We do not resell this data. It is provided to anyone for free for public use.

Technology

This site is built with open-source technologies:

  • Django - Python web framework
  • D3.js - Data visualization library
  • Tailwind CSS & DaisyUI - Modern UI framework
  • BeautifulSoup - HTML parsing for data collection
